## Key Mapping
+ ```SPC + ;```  : adds ; in EOL
+ ```CTRL + N``` : opens NERDTree
+ ```CTRL + P``` : File manager
+ ```CTRL + L``` : Loof for therm
+ ```CTRL + I``` : PlugInstall
